The Sierpinsky triangle is drawn by executing the L-system function
There are two modes of operation: manual and automatic.

In manual mode you can click anywhere in the triangle to step through the grammer.
In automatic mode the computer will do it for you.

The button "Run/Stop" switches between the two operation modes.
The speed slider let you change the running speed for the automatic mode.

The "depth" value defines the number of recursions the grammer will go through.
The minimum value is 1, the maximum value is 7.
The depth and speed values are activated by a click on the button "Restart".

The program gives an indication where in the grammer the program is.
This can be switched off and on again by hitting the button "(No) Execution"

Driving force: The L System: 

Alphabet: F, G, +, -

Axiom:    F+G+G

Rules:    F: F+G-F-G+F

          G: GG


Turtle actions:

          F,G: move forward

          +: turn left 120°

          -: turn right 120°
