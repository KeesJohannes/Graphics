The Sierpinsky triangle is drawn by executing the recursive function
There are two modes of operation: manual and automatic.

In manual mode you can click anywhere on the screen to step through the function.
In automatic mode the computer will do it for you.

The button "Run/Stop" switches between the two operation modes.

The speed slider let you change the running speed for the automatic mode: click "Stop", then change the speed then click the "Run" button.

The "depth" value defines the number of recursions the program will go through.
The minimum value is 1, the maximum value is 7.
It is activated by a click on the button "Restart".

In manual mode the program gives an indication at which level the program is.
The letters L, R and T indicate the left, right or top part of the diagram.

Driving force: recursive generator function

function* CutSierpinsky(pl,pr,pt,n) {

    let plr = halfbetween(pl,pr);

    let prt = halfbetween(pr,pt);

    let ptl = halfbetween(pt,pl);

    yield n; // 

    DrawTriangle(plr,prt,ptl);

    if (n>1) {

      yield* CutSierpinsky(pl,plr,ptl,n-1);

      yield* CutSierpinsky(plr,pr,prt,n-1);

      yield* CutSierpinsky(ptl,prt,pt,n-1);

    }

  }
